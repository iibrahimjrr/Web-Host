<h1>WebHost - Web Hosting Management System using PHP & MySQL</h1>

<p>
WebHost is a simple <b>web hosting management system</b> designed to simulate real-world hosting platforms. 
The application allows users to sign up and create their own websites for free, while providing administrators 
with full control over clients, orders, invoices, and support requests.
</p>

<p>
This project is built using core <b>PHP</b> and <b>MySQL</b>, and includes basic CMS (Content Management System) 
functionalities such as content pages, articles, search, FAQ, and contact forms. 
It is suitable as a learning project for understanding how hosting systems and client management platforms work.
</p>

<p>The following technologies were used in development:</p>
<ul>
  <li><p>PHP - Server-side scripting language used for building dynamic web applications.</p></li>
  <li><p>MySQL - Relational database management system used for storing application data.</p></li>
  <li><p>HTML5 - Used for structuring the web pages.</p></li>
  <li><p>CSS3 - Used for styling and layout.</p></li>
  <li><p>JavaScript - Used for basic interactivity on the client-side.</p></li>
</ul>

<h2>Key Features</h2>
<ul>
  <li><p>User registration and authentication system</p></li>
  <li><p>Free website hosting simulation</p></li>
  <li><p>Admin dashboard for managing clients, orders, and invoices</p></li>
  <li><p>Support request management system</p></li>
  <li><p>Built-in CMS (articles, pages, FAQ, search, contact forms)</p></li>
  <li><p>Customizable navigation menu controlled by admin</p></li>
</ul>

<h2>How It Works</h2>
<ol>
  <li><p>User registers and logs into the system</p></li>
  <li><p>User can create and manage their hosted website</p></li>
  <li><p>Admin monitors all users and manages system operations</p></li>
  <li><p>CMS allows content creation and customization</p></li>
</ol>

<h2>Project Purpose</h2>
<p>
The main goal of this project is to demonstrate how a web hosting platform works, including user management, 
service handling, and content management. It provides a simplified model of real hosting systems for educational purposes.
</p>

<h2>Installation Instructions</h2>
<ol>
  <li><p>Download the project and extract the files.</p></li>
  <li><p>Upload the project folder to your web server (or place it in your local server directory like XAMPP/htdocs).</p></li>
  <li><p>Set the following permissions:</p></li>
</ol>

<pre>
chmod 777 webhost
chmod 777 includes
</pre>

<ol start="4">
  <li><p>Open your browser and go to:</p></li>
</ol>

<pre>
http://localhost/webhost/first_run.php
</pre>

<pre>
first_run.php → first_run.php.bak
</pre>
